# Metadata: 

`merged_df.csv` 

**Description** 

This merged dataset contains individual grocery store locations in Chicago, along with their ZIP code, operational status, and the median household income of the area. 

- `store_name` (object): Name of the grocery store
- `address` (object): Street address of the store location
-`zip`(object): ZIP code here the store is located 
- `new_status` (object): Operational status of the store. Allow readers to identify if the store is still open and operating. 
- `median_income` (float): Median household income of the ZIP code area where the store is located
